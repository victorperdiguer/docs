## Claude Code

### Setup

1. Run the following command in your terminal to register the Uxia MCP server:
   ```bash
   claude mcp add uxia --scope user --transport http https://platform.uxia.app/api/mcp/mcp
   ```
   bash

Start a new Claude Code session in your terminal. In the list of MCP servers, select **Uxia**.

```bash
claude /mcp
```

1. Select **Authenticate**.
2. A browser window will open automatically. Sign in with your Uxia account to authorize access.

### Usage

Once connected, you can ask Claude perform actions in Uxia directly:

```plaintext
Create a new test thats tests my new app www.google.com 
```

## Claude Desktop

### Setup

1. In Claude Desktop, go to **Customize \> Connectors**.
2. Click the **\+** icon.
3. Click **Add custom connector**.
4. Enter **Uxia** as the name.
5. Paste the server URL:
   ```plaintext
   https://platform.uxia.app/api/mcp/mcp
   ```
6. Click **Add**.
7. On first use, a browser window will open automatically. Sign in with your Uxia account to authorize access.

If your Claude workspace is on a Team or Enterprise plan, you may need to ask your Claude workspace admin or owner to enable the Uxia MCP connector on your team. More information about this is available on [this Claude support page](https://support.claude.com/en/articles/11176164-use-connectors-to-extend-claude-s-capabilities).

### Usage

Once connected, you can ask Claude to talk to Uxia directly:

```plaintext
Create a new test thats tests my new app www.google.com 
```

Uxia's MCP server connects your workspace to Cursor so you can bring test results, transcripts, and UX insights into your coding and design conversations.

## Before you begin

You need:

- A Uxia workspace on a paid plan. MCP is not available on the Free Trial.
- The latest version of Cursor.
- Access to the Uxia workspace whose data you want to use.

<Steps>
  <Step title="Open Cursor's MCP settings">
    In Cursor, open **Settings → Tools & MCPs** and click **New MCP server**. Cursor opens your `mcp.json` file.
  </Step>
  <Step title="Add the Uxia server configuration">
    Paste this configuration into `mcp.json`:

    ```json
    {
      "mcpServers": {
        "Uxia": {
          "type": "http",
          "url": "https://platform.uxia.app/api/mcp/mcp",
          "headers": {}
        }
      }
    }
    ```

    If your `mcp.json` file already contains servers, add the `Uxia` entry inside the existing `mcpServers` object.
  </Step>
  <Step title="Authenticate Uxia">
    Return to **Settings → Tools & MCPs** in Cursor and enable **Uxia**. When Cursor first calls the server, a browser window opens. Sign in to Uxia and approve access to the workspace.
  </Step>
  <Step title="Start asking questions">
    Open Cursor Chat and ask Uxia-aware questions, such as:

    - _"List my five most recent Uxia tests."_
    - _"Summarize the insights from the checkout-flow test."_
    - _"What are the biggest usability issues in the latest website test?"_
  </Step>
</Steps>

<Note>
  Uxia connections use the same permissions as your Uxia account. You can revoke access at any time from **Settings → Integrations → MCP**.
</Note>

## What you can do with Uxia in Cursor

Once connected, Cursor can use your Uxia data as context while you work:

- Review test transcripts and research findings.
- Summarize insights and benchmark scores.
- Turn usability issues into implementation tasks.
- Compare feedback across tests to spot recurring patterns.

## Troubleshooting

If Uxia does not appear as connected in Cursor, confirm the URL is exactly `https://platform.uxia.app/api/mcp/mcp`, save `mcp.json`, then restart Cursor. If authentication does not open automatically, disable and re-enable the Uxia server from **Tools & MCPs**.

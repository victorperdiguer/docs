The **Human Insights** tab is the review-and-triage workspace for real test recordings and manually-reviewed sessions. It's less about AI synthesis and more about inspecting the sessions behind the results — the human side of a test, available once you've extended a test into a [Human Test](/testing/human-test).

## Overview

The tab opens with its own metrics summary for the reviewed sessions:

| Metric | What it shows |
| --- | --- |
| **Total testers** | How many human sessions were collected |
| **Success rate** | Share that reached the goal |
| **Drop-off rate** | Share that left before completing |
| **Misclick rate** | How often clicks didn't move the session forward |
| **Average duration** | Typical time spent per session |

## Statuses

Each session carries a status so you can triage what's worth reviewing:

| Status | Meaning |
| --- | --- |
| **Completed** | The tester finished the flow |
| **Abandoned** | The tester left before completing |
| **Missing questions** | The session finished but some post-test questions went unanswered |
| **Recording failed** | The session couldn't be captured |
